CSS PSUEDO CLASS..!

The CSS pseudo class allow you to style the dynamic states of an element such as hover, active and focus state, as well as elements that are existing in the document tree but can't be targeted via the use of other selectors without adding IDs oe classes t them, for example, targetting the first or last child.

A pseudo-class starts with a colon (:) .Its syntax can be given with :

selector: pseudo-class { property: value; }
